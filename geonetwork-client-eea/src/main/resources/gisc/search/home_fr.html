FIND INFORMATION ON GMES IN-SITU DATA RESOURCES AND PROVIDERS

The purpose of this catalogue is to register and provide an initial framework to provide access to in-situ data resources for GMES services. The scope of the catalogue is limited to in-situ data requirements for GMES pilot services.
The catalogue is based on Geonetwork which makes use of Inspire standards to facilitate sharing geographically referenced thematic data between different organizations. Meta-information on each in-situ data resource is provided according to the Inspire meta-information profile, and can be harvested from Inspire portals, organisational websites or hosted in Geonetwork. In this way the underpinning structure is flexible, decentralised, based on Inspire and shared.

If you are an in-situ data provider, then, once complete the data you provide should be described here. If you can't find your data, then contact the GISC project.

If you are a GMES service provider, then, once complete, the in-situ data you use should be catalogued and sometimes made available (linked to) from this portal.
