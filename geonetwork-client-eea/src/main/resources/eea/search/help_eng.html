Keyboard shortcuts

Application parameters

uuid={uuid} Popup the metadata with this uuid (eg search/?uuid=ed3263bc-8d0f-4047-9495-8120464b5121).
mode={0|1} Display the map panel by default if mode set to "1" (eg. search/?mode=1).
edit={id} Open the editor for the metadata identified by "id" according to user privileges.
